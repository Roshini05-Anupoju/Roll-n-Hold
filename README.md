# 🎲 Roll-n-Hold (Tenzies Game)
A fun React game where you roll dice until all values match.  
Click a disc to hold it and roll the rest.  
Built using **React + Vite**.

## 📽️Live Demo
- https://roshini05-anupoju.github.io/Roll-n-Hold/
## 🛠️ Technologies Used
- React
- Vite
- CSS
- GitHub Pages

## 🎯 How to Play
- Click a disc to hold its value.
- Roll only unheld discs.
- When all discs match → You win! 🎉

## 🧩 Features
- Hold/unhold discs  
- Roll only needed discs  
- Winning animation (confetti)  
- Clean, responsive UI
